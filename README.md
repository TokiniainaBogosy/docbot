# DocBot — Assistant documentaire intelligent

DocBot vous permet d'interroger vos documents PDF en langage naturel.
Uploadez vos fichiers, posez vos questions, obtenez des réponses instantanées
basées sur votre contenu — sans perdre de temps à chercher manuellement.

## Le problème
Les entreprises accumulent des documents essentiels — manuels, procédures,
contrats, fiches produits — que personne ne lit faute de temps. L'information
existe mais reste inaccessible.

## La solution
DocBot connecte un LLM à vos documents internes. Vos employés posent leurs
questions en français, DocBot trouve et synthétise les informations pertinentes
en quelques secondes.

## Fonctionnalités
- Upload de plusieurs PDFs simultanément
- Chat en langage naturel sur vos documents
- Affichage des sources utilisées pour chaque réponse
- Historique de conversation dans la session

## Stack technique
- Python · Streamlit · FAISS · Sentence Transformers · Groq API
